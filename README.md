# Number Guessing Game (Python)

## Description
A Python CLI game where the user guesses a randomly generated number.

## Features
- Random number generation
- Menu-driven using functions
- Tracks number of attempts
- Exit option

## Technologies Used
- Python
- Loops, Functions, Conditionals
- Random module

## How to Run
Run in Python interpreter:
# number-guessing-game
Python CLI project: number guessing game with functions

## Learning Outcome
- Practice with functions and loops  
- Understand random number generation  
- Logical thinking & user interaction
